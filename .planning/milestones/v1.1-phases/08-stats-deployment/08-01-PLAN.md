<feature>
  <name>Player Season Stats Data Layer</name>
  <files>src/season/playerStats.ts, src/season/playerStats.test.ts, src/season/quickSim.ts, src/season/season.ts</files>
  <behavior>
    PlayerSeasonStats interface:
    - Has fields: goals, assists, appearances, shots, shotsOnTarget, passes, passesCompleted, tacklesWon, tacklesAttempted, yellowCards, redCards, cleanSheets, minutesPlayed
    - createEmptySeasonStats() returns all zeros

    mergeMatchStats(season: PlayerSeasonStats, match: PlayerLogStats, minutesPlayed: number, teamConceded: number): PlayerSeasonStats:
    - Accumulates goals, assists, shots, shotsOnTarget, passes, passesCompleted, tacklesWon, tacklesAttempted from PlayerLogStats
    - Increments appearances by 1
    - Adds minutesPlayed
    - Reads yellowCards and redCards from match log entries (add these to PlayerLogStats first — see action)
    - cleanSheets: increments by 1 if player role is 'GK' AND teamConceded === 0

    mergeAllMatchStats(seasonStats: Map&lt;string, PlayerSeasonStats&gt;, matchStats: Map&lt;string, PlayerLogStats&gt;, goalsConceded: Record&lt;TeamId, number&gt;): Map&lt;string, PlayerSeasonStats&gt;:
    - For each player in matchStats, merges into seasonStats (creating new entry if absent)
    - minutesPlayed: 90 for all players (simplification — no sub tracking yet)
    - teamConceded: looks up goals conceded by the player's teamId

    QuickSimResult enhancement:
    - quickSimMatch returns { homeGoals, awayGoals, playerStats: Map&lt;string, PlayerLogStats&gt; }
    - Obtained via engine.gameLog.getPlayerStats() after match completes

    SeasonState enhancement:
    - Add playerSeasonStats: Map&lt;string, PlayerSeasonStats&gt; field
    - Initialized as empty Map in createSeason
    - Reset to empty Map in startNewSeason
    - Serializes automatically via existing MAP_TAG pattern (no serialize.ts changes)

    Cases:
    - mergeMatchStats with empty season stats + 2 goals, 1 assist -> goals: 2, assists: 1, appearances: 1
    - mergeMatchStats with existing stats (3 goals) + 1 goal -> goals: 4, appearances incremented
    - mergeAllMatchStats with two players from different teams, home concedes 0, away concedes 2 -> GK on home team gets cleanSheet, GK on away team does not
    - createEmptySeasonStats -> all fields are 0
    - quickSimMatch returns playerStats map with entries for all 22 players
  </behavior>
  <implementation>
    1. Create src/season/playerStats.ts with PlayerSeasonStats interface, createEmptySeasonStats(), mergeMatchStats(), mergeAllMatchStats()
    2. Add yellowCards and redCards fields to PlayerLogStats in gameLog.ts and populate them in getPlayerStats() by counting yellow_card/red_card entries per player
    3. Extend QuickSimResult to include playerStats field, extract from engine.gameLog.getPlayerStats() in quickSimMatch
    4. Add playerSeasonStats field to SeasonState, initialize in createSeason and startNewSeason as new Map()
    5. Use const-object pattern (not TS enum) per project conventions
  </implementation>
</feature>
