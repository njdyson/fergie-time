  <behavior>
    **Action Definitions (actions.ts):**
    - 8 actions defined: PASS_FORWARD, PASS_SAFE, DRIBBLE, SHOOT, HOLD_SHIELD, MOVE_TO_POSITION, PRESS, MAKE_RUN
    - Each action has 3-4 consideration functions (per research: "Start with 3-4 considerations per action maximum")
    - Each consideration function takes AgentContext and returns [0..1]
    - SHOOT considerations: distance-to-goal (sigmoid, drops sharply beyond 25m), shooting skill, clear line of sight
    - PASS_FORWARD considerations: teammate available forward, success probability, forward progress value
    - PASS_SAFE considerations: teammate available, success probability, under pressure (high pressure → higher safe pass score)
    - DRIBBLE considerations: space ahead, dribbling skill, no nearby defenders
    - HOLD_SHIELD considerations: under pressure, has ball, strength advantage
    - MOVE_TO_POSITION considerations: distance from formation anchor (closer = higher score), not already in position
    - PRESS considerations: distance to ball carrier, in defensive zone, aggression-related proximity
    - MAKE_RUN considerations: space to run into, forward position available, not offside zone
    - Hard disqualifiers: action returns 0 immediately if physically impossible (e.g., SHOOT without ball, PRESS when own team has ball)

    **Agent Evaluation (agent.ts):**
    - evaluateAction: multiplies all consideration scores (product), applies compensation factor, adds personality bonus, adds noise
    - Compensation factor formula: modFactor = 1 - 1/N, makeUp = (1-product) * modFactor, final = product + makeUp * product
    - selectAction: evaluates all actions, returns the one with highest score
    - With fixed seed + identical context + composure=1.0: selectAction returns same action >95% of the time
    - With fixed seed + identical context + composure=0.2: selectAction returns different actions >30% of the time

    **Personality Weight Matrix (personality.ts):**
    - PERSONALITY_WEIGHTS maps each ActionType to trait weights (per research Pattern 5)
    - dotProduct(weights, personality) computes the action-specific personality bonus
    - High directness: PASS_FORWARD bonus > PASS_SAFE bonus
    - High flair: DRIBBLE bonus > HOLD_SHIELD bonus
    - High aggression: PRESS bonus increased
    - High composure: HOLD_SHIELD bonus increased (via risk_appetite negative weight on HOLD_SHIELD being offset)
    - NOISE_SCALE constant: start at 0.08 (per research recommendation)
  </behavior>
