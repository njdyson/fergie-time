  <behavior>
    **Tackle Resolution (resolveTackle):**
    - Input: defender PlayerState, attacker PlayerState (ball carrier), rng
    - Output: TackleResult { success: boolean, foul: boolean }
    - Base success probability: `f(defender.tackling, attacker.dribbling)` — tackling advantage → higher success
    - Angle modifier: frontal tackle (0-45 degrees from front) → 1.0x, side (45-90) → 0.8x, behind (90-180) → 0.5x
    - Distance modifier: closer → higher success (within tackle range of ~2m)
    - Strength modifier: `defender.strength / (defender.strength + attacker.strength)` adds ±0.1
    - Foul probability: higher for behind tackles, lower for high-tackling defenders
    - Example: defender(tackling=0.8, strength=0.7) vs attacker(dribbling=0.4, strength=0.5), frontal → ~75% success
    - Example: defender(tackling=0.3, strength=0.4) vs attacker(dribbling=0.9, strength=0.8), behind → ~15% success

    **Shielding (isShielded):**
    - Input: ballCarrier PlayerState, challenger PlayerState
    - shieldRadius = BASE_SHIELD_RADIUS * ballCarrier.strength (BASE_SHIELD_RADIUS ~1.5m)
    - If challenger is within shieldRadius AND ball carrier is between ball and challenger → shielded = true
    - "Between" determined by: carrier-to-challenger vector dot carrier-to-ball vector > 0 (carrier facing the ball)
    - A strong player (strength=0.9) has shieldRadius ~1.35m, weak player (0.3) has ~0.45m

    **Aerial Contest (resolveAerialContest):**
    - Input: player1 PlayerState, player2 PlayerState, ball BallState, rng
    - Output: AerialResult { winnerId: string }
    - Jump height per player: BASE_JUMP * player.aerial (BASE_JUMP ~2.5m)
    - Can contest: ball.z <= jumpHeight AND player can reach ball ground position
    - Contest window: both players must be within ±0.3m of ball ground position
    - Winner: compare `aerial * 0.6 + strength * 0.3 + random * 0.1`
    - If only one player can reach: they win automatically

    **Spatial Grid:**
    - Divides pitch (105m x 68m) into cells (~10m x 10m → ~11 x 7 = 77 cells)
    - insert(player): places player in correct cell
    - query(position, radius): returns all players within radius (checks cell + adjacent cells)
    - clear(): empties all cells (called each tick before re-insert)
    - query(pos, 15m) with 22 players: O(1) per cell check, not O(22)
  </behavior>
